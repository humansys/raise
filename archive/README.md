# Archive

> Historical artifacts from previous framework versions

---

## Contents

| Directory | Description | Archived From |
|-----------|-------------|---------------|
| `agent/` | Old agent configuration | `.agent/` |
| `projects/` | Historical project artifacts | Legacy |
| `raise-kit/` | RaiSE Kit v1 (predecessor) | `.raise-kit-archived/` |
| `raise-v1/` | RaiSE v1 framework | `.raise-v1-archive/` |
| `specify/` | Spec-Kit tooling | `.specify/` |
| `specs-archived/` | Old specs internal archive | `specs/_archived/` |
| `specs-raise/` | Framework specs (now in framework/) | `specs/raise/` |

## Purpose

This directory preserves historical context and allows:

- **Archaeology**: Understanding how the framework evolved
- **Recovery**: Retrieving old patterns or approaches if needed
- **Reference**: Comparing old vs new implementations

## Policy

- **Read-only**: Do not modify archived content
- **No restoration**: If content needs to come back, create new in `work/`
- **Prune periodically**: Remove archives older than 2 major versions
