Welcome to the RaiSE documentation.

RaiSE combines a **lean methodology** with a **deterministic CLI toolkit** to make AI-assisted software engineering reliable, traceable, and repeatable.

## What you'll find here

- **[Getting Started](/docs/getting-started/)** — Install the CLI and run your first session
- **[Concepts](/docs/concepts/memory/)** — Memory, Skills, Governance, Knowledge Graph
- **[Guides](/docs/guides/first-story/)** — Your First Story, Setting Up a Project
- **[CLI Reference](/docs/cli/)** — Every command, flag, and option

## Quick install

```bash
pip install raise-cli
rai init
rai session start --project . --context
```
