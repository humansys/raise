# Issues

> Bugs and problems parked with evidence for later resolution.

## Format

Each issue: `ISSUE-NNN-short-description.md`

## Required Sections

| Section | Purpose |
|---------|---------|
| Problem | What's broken, expected vs actual |
| Evidence | Logs, analysis, reproduction steps |
| Root Cause | If known, or hypotheses |
| Workaround | How to work around it now |
| Investigation Needed | What to try next |

## Status

- `PARKED` — Not actively working, documented for later
- `INVESTIGATING` — Active root cause analysis
- `FIXING` — Solution identified, implementing
- `RESOLVED` — Fixed, keep for reference

## Priority

- `P0` — Blocking, no workaround
- `P1` — Painful, workaround exists but costly
- `P2` — Annoying, easy workaround
- `P3` — Minor, fix when convenient
