# S292.4 Scope: Cleanup session module

## In Scope

- Read session test files and identify muda using 4 heuristics
- Delete muda tests (constant assertions, mock-impl tests, magic counts, tautological tests)
- Verify `pytest tests/` passes after each deletion batch

## Out of Scope

- Refactoring source code in `src/.../session/`
- Adding new tests
- Fixing bugs discovered incidentally (→ parking lot)
- Changing test structure or organization

## Done Criteria

- [ ] Every deleted test fails at least one of the 4 heuristics (deletion criterion — PAT-E-499)
- [ ] All remaining tests pass (`pytest tests/`)
- [ ] No behavior tests deleted (only muda)
- [ ] Ratio measured and recorded as outcome (not used as gate)
