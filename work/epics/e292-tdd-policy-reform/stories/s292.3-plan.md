# S292.3 Plan: Cleanup hooks module (2.78x → <2.0x)

## Overview

Mechanical muda deletion across 6 test files in `tests/test_hooks/`. No new tests, no source changes.

**Baseline:**
- Source: 744 lines (hooks + hooks/builtin)
- Tests: 2070 lines
- Ratio: 2.78x (diagnostic — not a gate)

**Deletion criterion:** each deleted test must fail at least one of the 4 heuristics.
The ratio is measured as outcome; if it stays above 2.0x after honest application,
that signals either remaining muda to investigate or genuinely justified coverage.

## Muda Catalog

### test_events.py (281 lines)

**Pattern A: Tautological dataclass field tests** — assert that constructor assigns fields.
These fail only if Python's dataclass machinery breaks, not if our code breaks.
- `TestSessionStartEvent.test_payload_fields`
- `TestGraphBuildEvent.test_payload_fields`
- `TestBeforeSessionCloseEvent.test_payload_fields`
- `TestBeforeReleasePublishEvent.test_payload_fields`
- `TestHookResult.test_ok_result`, `test_abort_result`, `test_error_result`
- `TestEmitResult.test_aborted_result`

**Pattern B: Duplicate of parametrized TestAllEvents** — `TestAllEvents` already covers
`test_event_name`, `test_is_frozen`, `test_is_hook_event`, `test_has_timestamp` for all 11 events.
- `TestSessionStartEvent.test_event_name_is_session_start`, `test_is_hook_event_subclass`
- `TestGraphBuildEvent.test_event_name_is_graph_build`, `test_is_frozen`
- `TestBeforeSessionCloseEvent.test_event_name`, `test_is_frozen`, `test_is_hook_event`
- `TestBeforeReleasePublishEvent.test_event_name`, `test_is_frozen`
- `TestHookEvent.test_hook_event_is_frozen`, `test_hook_event_has_timestamp` (both in parametrized)

**Keep:** `TestHookEvent.test_hook_event_accepts_explicit_timestamp` (unique — not in parametrized),
`TestHookResult.test_is_frozen`, `TestEmitResult.test_default_not_aborted`, `TestEmitResult.test_is_frozen`,
all of `TestAllEvents`.

### test_protocol.py (90 lines)

**Pattern: Test fixture tests** — last 4 tests assert properties of `_CompliantHook`, not the protocol.
- `test_handle_returns_hook_result` — tests fixture behavior, not Protocol
- `test_events_is_list_of_strings` — asserts fixture constant
- `test_priority_is_int` — asserts fixture constant
- `test_default_priority_is_zero` — asserts fixture constant

**Keep:** 5 isinstance/non-isinstance checks.

### test_emitter.py (498 lines)

**Pattern: Trivial factory smoke tests** — `TestCreateEmitter` tests nothing behavioral.
- `test_create_emitter_returns_event_emitter` — `isinstance` on factory output
- `test_create_emitter_has_registry` — internal `_registry` attribute check
- `test_create_emitter_exported_from_package` — `callable()` import smoke test

**Keep:** All of `TestEmitterRegisterAndDispatch`, `TestEmitterErrorIsolation`,
`TestBeforeEventAbortFlow`, `TestEmitterWithRegistry`, `TestEmitterTimeout`, `TestE2EPipeline`.

### test_registry.py (189 lines)

**Pattern: Constant assertion** — `TestHookRegistryConstants.test_ep_hooks_value` asserts
`EP_HOOKS == "rai.hooks"`. String literal that only changes if we rename the entry point group.

**Keep:** All of `TestHookRegistryDiscover`, `TestHookRegistryGetHooksForEvent`, `TestHookRegistryManualRegister`.

### test_gate_bridge.py (196 lines)

**Pattern: Redundant protocol checks** in `TestProtocolConformance`:
- `test_high_priority` — constant assertion (`priority == 100`)
- `test_has_handle_method` — `callable(hook.handle)` already implied by `test_isinstance_check`

**Keep:** `test_isinstance_check`, `test_subscribes_to_before_events`, all of `TestGateExecution`,
`TestEntryPointDiscovery`.

### test_telemetry_hook.py (274 lines)

**Pattern: Redundant protocol conformance** in `TestProtocolConformance`:
- `test_has_events_classvar` — magic count `len == 9`, covered by `TestEventSubscription.test_subscribes_to_all_after_events`
- `test_has_priority_classvar` — constant assertion (`priority == 0`)
- `test_has_handle_method` — `callable()` implied by isinstance

**Pattern: Tautological discovery test**:
- `test_discovered_hook_is_functional` — checks `hook.events == TelemetryHook.events` and `hook.priority == 0`.
  If discovery works (covered by `test_registry_discovers_telemetry_hook`), attributes are necessarily correct.

**Keep:** `test_isinstance_check`, `TestEventSubscription.*`, `TestEventMapping.*`,
`TestErrorIsolation.*`, `test_registry_discovers_telemetry_hook`, `TestE2EIntegration.*`.
