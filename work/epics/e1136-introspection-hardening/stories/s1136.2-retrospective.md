Created `rai learn write` CLI command that validates learning records against the Pydantic model before writing to disk. Prevents schema drift (F2 from E1133 dogfood).

## Usage

```bash
# Write a learning record from YAML file
rai learn write record.yaml

# Validation error example
rai learn write bad.yaml
# Error: Invalid learning record: 1 validation error for LearningRecord
#   skill - Field required
```

## What's next
