# S1136.3: Auto-commit Learning Records

**Epic:** E1136 — Introspection Hardening
**Tracker:** RAISE-1238
**Type:** code
**Size:** XS

## User Story

As a developer using the LEARN protocol, I need learning records to be automatically included in git add during task commits, so that records persist across worktrees via merge and are not lost when a worktree is cleaned up.

## Acceptance Criteria (informal, XS)

- Learning records under `.raise/rai/learnings/` are included in `git add` during task commits
- Records survive worktree deletion by being committed to the story branch
- Records propagate to `release/2.4.0` (and other worktrees) via normal merge flow
