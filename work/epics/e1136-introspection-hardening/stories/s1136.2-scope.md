# S1136.2 Scope

## In Scope

1. `rai learn write` CLI command that accepts YAML stdin or file path
2. Validates against LearningRecord Pydantic model
3. Writes to `.raise/rai/learnings/{skill}/{work_id}/record.yaml`
4. Clear error messages on validation failure

## Out of Scope

- `rai learn push` (S1136.4)
- `rai learn list` / `rai learn show` (future)
- Auto-commit (S1136.3)

## Done Criteria

- [ ] `rai learn write --file record.yaml` validates and writes
- [ ] `rai learn write --skill X --work-id Y` writes from stdin
- [ ] Invalid records produce clear error with field name
- [ ] Tests cover happy path + validation failure
