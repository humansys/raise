# S1136.1 Retrospective

## What happened

Changed PRIME/LEARN from suggestive blockquotes to mandatory sections with inline schema in all 7 cognitive skills. Added Quality Checklist enforcement.

## Key change

Before: `> **PRIME**: Before Step 1, follow PRIME protocol...` (agent reads as optional)
After: `### PRIME (mandatory — do not skip)` with numbered steps (agent reads as required)

The inline YAML schema in LEARN eliminates the schema drift problem — the agent has the exact format right there in the skill, not in a separate aspect file it might not read.
