# S476.3: Create raise-dev skillset

## User Story
As a Python developer using RaiSE,
I want a `raise-dev` skillset with Python-specific verification commands
so that skills use explicit pytest/pyright/ruff commands instead of generic manifest lookups.

## Acceptance Criteria (informal, XS)
- `rai skill set create raise-dev` produces `.raise/skills/raise-dev/`
- Overlays exist for: rai-story-implement, rai-bugfix, rai-story-plan
- Each overlay replaces only the verification sections with explicit Python commands (pytest, pyright, ruff)
- Overlays are minimal -- only the sections that differ from base skills

## Scope
- Skillset creation via CLI or manual structure
- 3 overlay files in `.raise/skills/raise-dev/`
- No changes to base skills (those are manifest-first from S476.2)
