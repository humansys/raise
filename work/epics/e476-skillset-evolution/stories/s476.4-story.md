# S476.4: Create raise-dev-ts example

## User Story
As a TypeScript developer using RaiSE,
I want a `raise-dev-ts` skillset with TypeScript-specific verification commands
so that skills use explicit vitest/tsc/eslint commands instead of generic manifest lookups.

## Acceptance Criteria (informal, XS)
- `.raise/skills/raise-dev-ts/` directory exists with overlay files
- Overlays exist for: rai-story-implement, rai-bugfix, rai-story-plan
- Each overlay replaces only the verification sections with explicit TypeScript commands (vitest, tsc, eslint)
- Overlays mirror the raise-dev structure but with TypeScript tooling
- No changes to base skills or to the raise-dev skillset

## Scope
- Skillset creation (manual structure, mirroring raise-dev)
- 3 overlay files in `.raise/skills/raise-dev-ts/`
- Demonstrates the skillset pattern works for non-Python projects
