# E350: Rai Experience Portability — Scope

## Objective
Make the Rai experience fully reproducible for any developer joining a RaiSE project, so that `rai init` + `rai graph build` gives them complete working context.

## In Scope
- Artifact inventory and classification (universal/project/personal)
- Extract distributable context into rai-cli distribution pipeline
- Enhance `rai init` to generate agent context from `.raise/` canonical source
- Fix MEMORY.md regeneration on `rai graph build`
- Sanitize `.gitignore` for parallel multi-developer work
- `rai init` auto-adds `personal/` to `.gitignore`

## Out of Scope
- IDE plugins beyond Claude Code
- Cross-machine state migration
- Personal calibration sharing
