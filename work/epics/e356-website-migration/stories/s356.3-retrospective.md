## What I Learned
1. **wrangler-action@v3 is the clean path** — GitHub Actions with wrangler-action@v3 handles
   Cloudflare Pages deployment cleanly from a monorepo. No custom scripts needed.
2. **Path filters for monorepo scoping** — `on.push.paths` filters are essential to avoid
   triggering deploys on unrelated changes in a monorepo.
