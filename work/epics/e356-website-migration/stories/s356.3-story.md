# S356.3: Cloudflare Pages deployment

## User Story

As a RaiSE maintainer, I want the website to deploy from raise-commons via Cloudflare Pages, so that site updates are published automatically when changes merge to dev/main.

## Acceptance Criteria

- AC1: GitHub Actions workflow exists for deploying site/ to Cloudflare Pages

- AC3: Deploy instructions documented for manual fallback
- AC4: D1 database migration documented
