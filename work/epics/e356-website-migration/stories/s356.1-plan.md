### T1: Copy site/ from raise-gtm (S)

**Action:** rsync `~/Code/raise-gtm/site/` to `./site/` excluding build artifacts.

```bash

  ~/Code/raise-gtm/site/ ./site/
```

**Verification:**
- `site/` directory exists with source files (astro.config.mjs, package.json, src/, public/)

**Dependencies:** None
