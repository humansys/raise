# S356.1 Scope

## In Scope

- Copy `site/` directory from `~/Code/raise-gtm/site/` to `raise-commons/site/`
- Exclude `node_modules/`, `dist/`, `.astro/` from the copy
- Add `.gitignore` entries for `site/node_modules`, `site/dist`, `site/.astro`
- Verify the site builds with `npm install && npm run build` inside `site/`

## Out of Scope

- Content changes (landing page copy, pricing, blog) — that is S356.2
- Cloudflare Pages deployment reconfiguration — that is S356.3
- Any new blog posts — that is S356.4
- Removing or archiving the site in raise-gtm

## Done Criteria

- [ ] `site/` exists in raise-commons with all source files
- [ ] `npm run build` succeeds in `site/`
- [ ] `node_modules`, `dist`, and `.astro` are gitignored
