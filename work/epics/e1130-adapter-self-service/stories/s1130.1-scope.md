# S1130.1 Scope: Confluence Discovery Service

## In Scope

- `ConfluenceSpaceMap` frozen dataclass: `spaces: list[SpaceInfo]`, `top_level_pages: dict[str, list[PageSummary]]`
- `ConfluenceDiscovery` class wrapping `ConfluenceClient`
- `discover(space_key=None)` → full or filtered `ConfluenceSpaceMap`
- Error propagation: auth errors, not-found, API errors from client
- Unit tests with mocked `ConfluenceClient`

## Out of Scope

- New methods on `ConfluenceClient` (already has `get_spaces()`, `get_page_children()`)
- CLI command for discovery (D1: internal service only)
- Labels or content discovery (not needed for config generator)
- Integration tests against live Confluence (deferred to S1130.3 doctor)

## Done Criteria

1. `ConfluenceDiscovery.discover()` returns `ConfluenceSpaceMap` with spaces and top-level pages
2. `ConfluenceDiscovery.discover(space_key="X")` returns filtered map for one space
3. Error cases covered: auth failure, not found, empty space
4. All tests pass, types check, linting clean
5. Module: `packages/raise-cli/src/raise_cli/adapters/confluence_discovery.py`
