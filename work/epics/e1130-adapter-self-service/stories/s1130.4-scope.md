# S1130.4 Scope: Config Generator — Confluence

## In Scope

- `generate_confluence_config(space_map, selected_space, instance_url, routing=None) → dict`
- Pure function — no side effects, no file I/O
- Output matches `ConfluenceConfig.from_dict()` schema (validated in tests)
- Default routing for common artifact types (adr, developer) when no routing provided
- Error on invalid selected_space

## Out of Scope

- YAML serialization or file writing (S1130.6 skill handles that)
- Interactive space selection (S1130.6 skill handles that)
- Multi-instance generation (single instance per call, composable)
- Confluence client or discovery changes

## Done Criteria

1. `generate_confluence_config()` returns dict that passes `ConfluenceConfig.from_dict()` validation
2. Generated config includes selected space, URL, and instance name
3. Routing entries included when provided, sensible defaults when not
4. Error raised for space not in discovery map
5. All tests pass, types check, linting clean
6. Module: `packages/raise-cli/src/raise_cli/adapters/confluence_config_gen.py`
