## Key Patterns

### P1: D2 (Memory) is the biggest systemic gap
Only 5/40 skills consult the graph (discover, docs-update, project-create, project-onboard, story-implement). The graph exists and has valuable data (patterns, guardrails, decisions), but 87% of skills ignore it.

### P2: D1 (Guardrails) maxes at 2
No skill achieves score 3 (runtime enforcement). The best we have is CLI gate checks (node counts, file existence). Governance hooks exist in rai-agent but are disabled (RAISE-1007).

### P3: Review skills are structurally weak
quality-review and architecture-review are the lowest-scoring lifecycle skills. They're purely inferential — no CLI extraction, no graph consultation, no verification. Ironic: the skills that judge quality have the least quality infrastructure.

### P4: Knowledge pipeline is understructured
5 knowledge skills (discover, extract, validate, diff, run) all score 5-6. They were built as E3 MVPs and haven't matured. The "validate" skill doesn't enforce its own validation gates.

### P5: UX is the strongest dimension
Mean 2.7, all skills ≥2. ADR-040 has been effective for structural consistency. This is the foundation to build on.

### P6: There's a maturity cliff between lifecycle and utility skills
Lifecycle skills (session, epic, story) average 10.2 total. Utility skills average 8.1 total. The lifecycle chain has had more iteration cycles.

## Recommendations for S14.2 (Gap Analysis)

Priority gaps to address with stories:

1. **CRITICAL: Enable governance hooks** (RAISE-1007) — unblocks D1 for all skills
2. **HIGH: Default memory injection** — moves D2 from 0.4 to ~2.0 for all skills via runtime, not per-skill changes
3. **HIGH: Harden review skills** — quality-review and architecture-review need CLI extraction + graph context
4. **MEDIUM: Knowledge pipeline maturity** — 5 skills need gates, graph integration, typed outputs
5. **MEDIUM: Typed skill metadata** — evolve inputs/outputs from strings to parseable schema
6. **LOW: Life/reporting skills** — low scores are by design (coaching, not enforcement)
