**Story:** S1134.4 | **Size:** S | **Date:** 2026-04-02

## Tasks

### T1: Write validation script (S)

**Description:** Create `work/epics/e1134-cc-alignment/validation/validate.py` — a single-file Python script (stdlib + PyYAML) that:

1. Discovers all `.claude/skills/*/SKILL.md` files via glob
2. Parses YAML frontmatter (split on `---` markers, `yaml.safe_load`)
3. Collects metrics across 3 dimensions:
   - **Dimension 1 (Descriptions):** char count, verb-first check, total budget for auto-invocable skills
   - **Dimension 2 (allowed-tools):** presence, bare `Bash` vs glob patterns
   - **Dimension 3 (Invocation control):** `disable-model-invocation` presence and count
4. Hardcodes "before" baseline from epic design Gemba audit
5. Formats markdown report with before/after summary table + per-dimension detail
6. Writes report to `work/epics/e1134-cc-alignment/validation/report.md`

**Files to create:**
- `work/epics/e1134-cc-alignment/validation/validate.py`

**Verification:**
```bash
# Script runs without errors
python3 work/epics/e1134-cc-alignment/validation/validate.py

# Report file created
test -f work/epics/e1134-cc-alignment/validation/report.md && echo "OK"
```

**Dependencies:** None (reads files created by S1134.1, S1134.2, S1134.3 — all done)
