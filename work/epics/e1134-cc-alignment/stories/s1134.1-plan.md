### T1: Rewrite all 35 descriptions (M)

**Description:** Read each SKILL.md, understand the skill's purpose from its content, write a new description following D1 pattern (verb-first, <100 chars, trigger phrases).

**Files to modify:** All 35 `.claude/skills/*/SKILL.md`

**Approach:** Process by lifecycle group for consistency. Read current description + first section of skill body to understand purpose. Write new description.

**Verification:**
```bash
# All descriptions under 100 chars
for f in .claude/skills/*/SKILL.md; do
  desc=$(python3 -c "import yaml; d=yaml.safe_load(open('$f').read().split('---')[1]); print(d.get('description',''))")
  chars=$(echo -n "$desc" | wc -c)
  skill=$(basename $(dirname $f))
  [ $chars -gt 100 ] && echo "OVER: $skill ($chars chars)"
done

# All start with a verb (capital letter)
for f in .claude/skills/*/SKILL.md; do
  desc=$(python3 -c "import yaml; d=yaml.safe_load(open('$f').read().split('---')[1]); print(d.get('description',''))")
  first=$(echo "$desc" | head -c1)
  skill=$(basename $(dirname $f))
  [[ ! "$first" =~ [A-Z] ]] && echo "NO VERB: $skill — starts with '$first'"
done

# YAML valid
for f in .claude/skills/*/SKILL.md; do
  python3 -c "import yaml; yaml.safe_load(open('$f').read().split('---')[1])" 2>&1 || echo "FAIL: $f"
done
```
