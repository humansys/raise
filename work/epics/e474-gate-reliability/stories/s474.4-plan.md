## T1: Hook Module -- `src/raise_cli/gates/hook.py`

**Goal:** Create the Python hook module that loads manifest and runs lint, format, and type-check commands sequentially.

**TDD Sequence:**

1. **RED:** Test `main()` returns 0 when all three commands pass (mock `load_manifest` and `subprocess.run`)
2. **GREEN:** Implement `main()` that loads manifest, runs `lint_command`, `format_command`, `type_check_command` via `subprocess.run()`
3. **RED:** Test exit 1 when lint fails, output shows which check failed
4. **GREEN:** Add per-check pass/fail reporting with check name
5. **RED:** Test unconfigured command (manifest key is None) is skipped gracefully
6. **GREEN:** Skip when `getattr(manifest.project, key, None)` returns None
7. **RED:** Test missing manifest returns exit 1
8. **GREEN:** Handle `load_manifest()` returning None
9. **RED:** Test that `test_command` is never executed (assert subprocess not called with test command)
10. **GREEN:** Hardcode the three command keys; `test_command` is excluded by design

**Implementation Notes:**
- Reuse `load_manifest(Path.cwd())` from `raise_cli.onboarding.manifest`
- Use same `command.split()` + `subprocess.run()` pattern as `_runner.py`
- Module must be runnable as `python -m raise_cli.gates.hook` (add `if __name__ == "__main__"` block)
- Print clear output: `[PASS] lint`, `[FAIL] format`, etc.
- `sys.exit(0)` if all pass, `sys.exit(1)` if any fail

## T2: CLI Commands -- `rai gate install-hook` / `rai gate uninstall-hook`

**Goal:** Add two subcommands to `gate_app` that manage the `.git/hooks/pre-commit` shim.

**TDD Sequence:**

1. **RED:** Test `install-hook` creates `.git/hooks/pre-commit` with correct shim content and executable permissions
2. **GREEN:** Implement `install_hook` command that writes shim, `chmod +x`
3. **RED:** Test `install-hook` refuses to overwrite existing hook (exits 1 with warning)
4. **GREEN:** Check existence before write, exit with message
5. **RED:** Test `install-hook --force` overwrites existing hook
6. **GREEN:** Add `--force` flag that bypasses existence check
7. **RED:** Test `uninstall-hook` removes rai-installed hook (marker comment present)
8. **GREEN:** Read file, check for `# Installed by: rai gate install-hook` marker, delete
9. **RED:** Test `uninstall-hook` refuses to remove non-rai hook (exits 1 with warning)
10. **GREEN:** Check marker before removal, warn if not found

**Shim content:**
```bash

# Remove with:  rai gate uninstall-hook
```
