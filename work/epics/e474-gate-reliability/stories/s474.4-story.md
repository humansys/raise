# S474.4: Pre-commit Hook for Lint+Format+Types

## Summary

Create a pre-commit hook configuration that runs lint, format, and type checks before each commit, preventing code that would fail CI from being committed.

## Acceptance Criteria

1. Pre-commit hook runs `ruff check`, `ruff format --check`, and `pyright` before each commit
2. Hook reads commands from `.raise/manifest.yaml` (manifest-driven)
3. Hook is opt-in: developer runs a setup command to install it
4. Commit is blocked if any check fails, with clear error output
5. A bypass escape hatch exists for exceptional cases (e.g., `--no-verify`)
