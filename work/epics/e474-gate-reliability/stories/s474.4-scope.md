# S474.4 Scope: Pre-commit Hook for Lint+Format+Types

## In Scope

- Pre-commit hook script that runs lint, format, and type checks
- Manifest-driven: commands sourced from `.raise/manifest.yaml` (`lint_command`, `format_command`, `type_check_command`)
- Setup command/script to install the hook into `.git/hooks/pre-commit`
- Clear error output showing which check failed and how to fix it
- Documentation of the setup process (inline in script or help flag)

## Out of Scope

- Test execution in pre-commit (too slow for commit-time)
- Automatic fixing (hook only checks, does not modify files)
- Forced installation (must remain opt-in)
- Integration with external pre-commit frameworks (e.g., pre-commit.com)

## Technical Approach

1. Shell script at `tools/pre-commit-hook.sh` (or similar) that:
   - Reads commands from `.raise/manifest.yaml` using simple parsing
   - Runs each check sequentially, collecting results
   - Reports pass/fail per check with actionable messages
   - Exits non-zero if any check fails

2. Setup script/command (`rai gate install-hook` or `make install-hooks`) that:
   - Copies/symlinks the hook into `.git/hooks/pre-commit`
   - Makes it executable
   - Confirms installation

## Dependencies

- S474.2 (FormatGate) — completed
- `.raise/manifest.yaml` — exists with all required commands
