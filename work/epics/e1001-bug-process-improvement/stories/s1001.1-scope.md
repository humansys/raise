# S1001.1: Jira Infrastructure — Scope

## In Scope
- Create 4 custom select fields in Jira admin (Bug Type, Severity, Origin, Qualifier)
- Each field with exact constrained values per design.md
- Activate bug workflow: Open → Triage → Diagnose → Fix → Verify → Retro → Done
- Validate with a test bug (create, classify, transition through full workflow)

## Out of Scope
- Automation (webhooks, scripts, post-functions)
- Changes to Story or Task issue types
- Bulk retroactive classification (that's S1001.3)
- rai-bugfix skill changes (that's S1001.2)

## Done Criteria
- [ ] 4 custom fields visible on Bug issue type with constrained values
- [ ] Bug workflow active with 7 statuses (Open → Triage → Diagnose → Fix → Verify → Retro → Done)
- [ ] Test bug created, classified in 4 dimensions, transitioned through full workflow
- [ ] Field names documented for S1001.2 to reference
