# S1001.1: Jira Infrastructure — Scope

## In Scope
- Create 4 custom select fields in Jira admin (Bug Type, Severity, Origin, Qualifier)
- Each field with exact constrained values per design.md
- Activate bug workflow: Open → Triaged → Diagnose → Fix → Verify → Retrospective → Done
- Validate with a real bug (RAISE-1152, classify, transition through workflow)

## Out of Scope
- Automation (webhooks, scripts, post-functions)
- Changes to Story or Task issue types
- Bulk retroactive classification (that's S1001.3)
- rai-bugfix skill changes (that's S1001.2)

## Done Criteria
- [x] 4 custom fields visible on Bug issue type with constrained values
- [x] Bug workflow active with 7 statuses (Open → Triaged → Diagnose → Fix → Verify → Retrospective → Done)
- [x] RAISE-1152 created, classified in 4 dimensions, transitioned Triaged→Diagnose→Open→Triaged — fields persisted
- [x] Field names documented in design.md for S1001.2 to reference
