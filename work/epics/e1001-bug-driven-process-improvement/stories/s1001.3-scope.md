# S1001.3 Scope — Dogfood Bug Classification

## In Scope
- Process all 2.3.x bugs through `/rai-bugfix` (7-phase workflow)
- Classify each bug with 4 dimensions (Bug Type, Severity, Origin, Qualifier)
- Extract process improvement from each bug (resolved or open)
- Update Jira custom fields for each bug
- Pattern analysis: dominant Type x Origin x Qualifier combinations
- Summary report with top 3+ process improvements

## Out of Scope
- Fixing unresolved bugs (classification only, fixes are separate stories)
- Bugs from versions other than 2.3.x
- Dashboard/JQL filters (that's S1001.4)
- Changes to the `/rai-bugfix` skill itself

## Done Criteria
- [ ] Every 2.3.x bug has 4 custom fields filled in Jira
- [ ] Process improvement extracted and recorded for each bug
- [ ] Pattern analysis completed (cross-dimensional)
- [ ] Summary report published
- [ ] Top 3+ concrete process improvements identified
