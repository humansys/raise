> **Revised:** v3 — artifact type as input restored (domain intelligence).

## In Scope

- `rai docs publish <artifact-type>` — resolve path by convention (`governance/{type}.md`), read file, publish via DocumentationTarget
- `rai docs get <page-id>` — retrieve page content as markdown from remote
- `rai docs search <query>` — search pages on remote target, compact output
- Generic `_resolve.py` — refactor `_adapter_resolve.py` to parametrized resolver, both backlog and docs use it
- Delete `_adapter_resolve.py` after migration
- Registration in main CLI (`main.py`)
- Tests for all commands + generic resolver

## Out of Scope

- McpConfluenceAdapter (S301.5)
- Platform config (spaces, parent pages) — adapter's responsibility (S301.5)
- GovernanceSchemaProvider integration (doesn't exist yet)
- Batch publish
- `rai docs delete`
