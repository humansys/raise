# S301.4: `rai docs` CLI Group — Scope

## In Scope

- `rai docs publish <path>` — publish a local file via DocumentationTarget
- `rai docs get <identifier>` — retrieve page content as markdown
- `rai docs search <query>` — search pages, compact output
- `_docs_resolve.py` — resolve DocumentationTarget from entry points (mirrors `_adapter_resolve.py`)
- Registration in main CLI (`main.py`)
- Tests for all commands + resolver

## Out of Scope

- McpConfluenceAdapter (S301.5, separate story)
- Doc type detection / metadata inference from file path
- Batch publish
- `rai docs delete`

## Done Criteria

- [ ] `rai docs --help` shows publish, get, search commands
- [ ] `resolve_docs_target()` auto-detects from `rai.docs.targets` entry points
- [ ] All commands delegate to DocumentationTarget protocol
- [ ] Compact output (~200 tokens per operation)
- [ ] Tests pass (TDD, red-green-refactor)
- [ ] Type checks pass (pyright)
- [ ] Lint passes (ruff)
