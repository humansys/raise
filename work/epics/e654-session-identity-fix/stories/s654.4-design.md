# S654.4 Design — Context Bundle & Orientation

## Decisions

| # | Decision | Choice | Rationale |
|---|----------|--------|-----------|
| D1 | Data source | Shared index via `read_session_entries(prefix)` | Cross-environment history |
| D2 | Fallback | Personal index if shared index empty | Backward compat for unmigrated repos |
| D3 | Display format | `ID: name (type, duration)` | Human-readable, shows names |
| D4 | Active session | Show name from `read_active_session()` pointer | Session identity visible in bundle |

## Changes

### `session/bundle.py` — `_format_recent_sessions()`

- Add `developer_prefix: str | None` parameter
- If prefix available: read from shared index via `read_session_entries()`
- If prefix unavailable or shared index empty: fall back to personal index (existing behavior)
- Format entries with name, type, and duration

### `session/bundle.py` — `assemble_context_bundle()`

- Pass `profile.get_pattern_prefix()` to `_format_recent_sessions()`
- Include active session name in session line
