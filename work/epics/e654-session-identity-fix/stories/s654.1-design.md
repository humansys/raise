# S654.1 Design — Session ID Model & Prefix Registry

## Decisions

| # | Decision | Choice | Rationale |
|---|----------|--------|-----------|
| D1 | Session ID generation | Pure function `generate_session_id(prefix, now)` | 3 lines, no state, testable with injected time |
| D2 | PrefixRegistry | Pydantic model (`PrefixRegistry` + `PrefixEntry`) | Project rule: Pydantic for all data structures |
| D3 | Module placement | New `session/identity.py` for ID generation, `session/prefix.py` for registry | Clear separation: identity ≠ memory, registry ≠ generation |
| D4 | Resolver strategy | Recognize both old and new formats as valid, no conversion | Migration is S654.5's job |

## Components

### `session/identity.py` (new)

```python
def generate_session_id(prefix: str, now: datetime | None = None) -> str:
    """Generate timestamp-based session ID: SES-{prefix}-{YYYYMMDD}T{HHMM}."""
    ts = now or datetime.now()
    return f"SES-{prefix}-{ts.strftime('%Y%m%dT%H%M')}"
```

### `session/prefix.py` (new)

```python
class PrefixEntry(BaseModel, frozen=True):
    name: str
    registered: date

class PrefixRegistry(BaseModel):
    prefixes: dict[str, PrefixEntry]

    def register(self, prefix: str, name: str) -> str:
        """Register prefix. Returns actual prefix (extended if collision)."""

    def resolve_collision(self, prefix: str, name: str) -> str:
        """Suggest extended prefix on collision (E → EO)."""

    @classmethod
    def load(cls, path: Path) -> PrefixRegistry:
        """Load from prefixes.yaml, or empty registry if not found."""

    def save(self, path: Path) -> None:
        """Write to prefixes.yaml."""
```

### `session/resolver.py` (modified)

Update `_normalize_session_id()`:
- New format `SES-{prefix}-{timestamp}` → passthrough (uppercase)
- Old format `SES-NNN` → passthrough (uppercase)
- Numeric `177` → `SES-177` (backward compat)
- Path traversal → reject
