# S654.2 Design — Shared Session Index

## Decisions

| # | Decision | Choice | Rationale |
|---|----------|--------|-----------|
| D1 | Module | New `session/index.py` | Shared index is session concept, not memory. Separate from `memory/writer.py` |
| D2 | Entry model | `SessionIndexEntry` Pydantic frozen | Project rule. Entries are immutable once written |
| D3 | Active pointer | Plain text file containing session ID | Single value, no structure needed |
| D4 | Wiring | Functions only, no CLI integration | S654.3 connects to CLI commands |

## Components

### `session/index.py` (new)

```python
class SessionIndexEntry(BaseModel, frozen=True):
    id: str                          # S-E-260322-1430
    name: str                        # "gemba session identity"
    started: datetime
    closed: datetime | None = None
    type: str = "feature"
    summary: str = ""
    outcomes: list[str] = []
    branch: str = ""

def write_session_entry(
    prefix: str, entry: SessionIndexEntry, project_root: Path | None = None
) -> Path:
    """Append entry to .raise/rai/sessions/{prefix}/index.jsonl."""

def read_session_entries(
    prefix: str, project_root: Path | None = None
) -> list[SessionIndexEntry]:
    """Read all entries from .raise/rai/sessions/{prefix}/index.jsonl."""

def write_active_session(
    session_id: str, project_root: Path | None = None
) -> None:
    """Write session ID to .raise/rai/personal/active-session."""

def read_active_session(
    project_root: Path | None = None
) -> str | None:
    """Read session ID from .raise/rai/personal/active-session. None if not found."""

def clear_active_session(
    project_root: Path | None = None
) -> None:
    """Remove the active session pointer."""
```
