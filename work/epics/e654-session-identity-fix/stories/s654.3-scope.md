# S654.3 Scope — CLI Adaptation

## In Scope

- `rai session start`: use `generate_session_id()`, write active pointer, require `--name`
- `rai session close`: write to shared index via `write_session_entry()`, clear pointer
- `rai session list`: read from shared index, display names
- Wire `session/close.py` to use shared index (replace or supplement personal index)
- Prefix auto-registration on first session start

## Out of Scope

- Context bundle changes (S654.4)
- Migration of old sessions (S654.5)
- Git auto-commit on close (parking lot)

## Done Criteria

- [ ] `rai session start --name "..."` generates S-{P}-{YYMMDD}-{HHMM} ID
- [ ] `rai session close` appends to shared index
- [ ] Active pointer written on start, cleared on close
- [ ] Prefix auto-registered on first use
- [ ] Tests pass, types clean, lint clean
