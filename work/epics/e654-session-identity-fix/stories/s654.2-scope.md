# S654.2 Scope — Shared Session Index

## In Scope

- Session index entry model (Pydantic): id, name, started, closed, type, summary, outcomes, branch
- Write session entry to `.raise/rai/sessions/{prefix}/index.jsonl` (append JSONL)
- Read session entries from shared index (load all, load latest N)
- Active session pointer: read/write `.raise/rai/personal/active-session`
- Directory auto-creation on first write

## Out of Scope

- CLI command integration (S654.3)
- Context bundle reads (S654.4)
- Migration of old index (S654.5)
- Git commit automation (deferred to parking lot)
