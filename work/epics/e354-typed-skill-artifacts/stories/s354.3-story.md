# Story: S354.3 — Graph Ingestion

> **Epic:** E354 — Typed Skill Artifacts
> **Branch:** `story/s354.3/graph-ingestion`
> **Size:** S
> **Depends:** S354.1 (base artifact model)

## Summary

Add `load_artifacts()` to `GraphBuilder` so that `rai graph build` ingests `.raise/artifacts/*.yaml` as graph nodes with ref edges.

## Acceptance Criteria

1. `GraphBuilder.load_artifacts()` reads all YAML from `.raise/artifacts/`
2. Each artifact becomes a `GraphNode` with appropriate type and metadata
3. `refs` fields become edges to referenced nodes (backlog items, epics)
4. `rai graph build` includes artifact nodes in the graph output
5. Empty or missing `.raise/artifacts/` produces no errors
