# Story: S354.1 — Base Artifact Model + Storage

> **Epic:** E354 — Typed Skill Artifacts
> **Branch:** `story/s354.1/base-artifact-model`
> **Size:** S
> **Depends:** None (foundation story)

## Summary

Create the `SkillArtifact` base Pydantic model, `ArtifactType` enum, and YAML reader/writer for `.raise/artifacts/`. This is the foundation all other E354 stories build on.

## Acceptance Criteria

1. `SkillArtifact` base model with common fields (type, version, skill, created, story, epic, content, refs)
2. `ArtifactType` enum with `story-design` as first value
3. YAML writer: serialize artifact to `.raise/artifacts/{id}-{type}.yaml`
4. YAML reader: deserialize YAML back to validated Pydantic model
5. Round-trip test: write → read → validate → fields match
6. New module at `src/rai_cli/artifacts/` with `__init__.py`, `models.py`, `writer.py`, `reader.py`
