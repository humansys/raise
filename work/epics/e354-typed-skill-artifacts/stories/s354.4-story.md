# Story: S354.4 — Doc Generation

> **Epic:** E354 — Typed Skill Artifacts
> **Branch:** `story/s354.4/doc-generation`
> **Size:** S
> **Depends:** S354.2 (story-design schema)

## Summary

Render StoryDesignArtifact YAML into human-readable Markdown, written to `work/docs/`. Generated docs are disposable derivatives of the YAML source of truth.

## Acceptance Criteria

1. `render_artifact(artifact) → str` produces valid Markdown
2. Markdown includes: title, metadata, summary, AC, integration points, decisions
3. `write_doc(artifact, project_root) → Path` writes to `work/docs/{id}-{type}.md`
4. Round-trip check: no substantive information lost between YAML and generated Markdown
5. Missing optional fields (decisions, integration_points) handled gracefully
