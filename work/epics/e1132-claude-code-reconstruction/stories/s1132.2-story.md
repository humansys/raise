## User Story

**As a** RaiSE framework developer,
**I want** deep understanding of Claude Code's extension points (skills, hooks, tools, commands),
**So that** I can build RaiSE extensions that correctly integrate with Claude Code's contracts and lifecycle.

```gherkin

  Scenario: Commands registration documented

```

## Method

ADR-016 Phase 2 — Targeted Deep Dives. For each module:
1. Read key files (index.ts, types, main implementation)
2. Trace lifecycle from registration to execution
3. Document finding with RaiSE impact
4. Confidence: Alta (source code is authoritative)
