# S346.2: Language-Agnostic Quality Review — Design

## Complexity: Simple/Moderate (1 component, text editing)

## Problem

quality-review hardcodes Python patterns (`*.py`, `type: ignore`, `cast()`, `except Exception`, `__all__`, pytest), making it useless for non-Python projects.

## Value

QR produces meaningful semantic bug detection for Python, TypeScript, and C#/.NET — critical for multi-stack adoption (Kurigage: Symphony/PHP, .NET, Flutter).

## Approach

Replace hardcoded Python references with a language-adaptive structure:

1. **Add Step 0 (Language Detection):** Read `.raise/manifest.yaml` for project languages. Fallback: scan file extensions of changed files.
2. **Adapt Step 1 (File Filter):** Use detected language extensions instead of hardcoded `*.py`.
3. **Restructure Step 2 (Semantic Checks):** Organize by language with equivalent checks:
   - Python: `type: ignore`, `cast()`, `except Exception`, `raise X from exc`, `__all__`
   - TypeScript: `as` casts, `any` types, unhandled promises, `@ts-ignore`
   - C#: null safety (`!` operator), unchecked casts, `IDisposable` without `using`
   - Universal: inverted conditionals, off-by-one, wrong variable, unhandled edge cases
4. **Keep Step 3 (Test Heuristics):** 7 heuristics are universal — only change framework examples (pytest → jest/xunit).
5. **Adapt Step 4 (API Surface):** Language-appropriate idioms (`__all__` → `export`, `public` visibility).

## Components

| Component | Change |
|-----------|--------|
| `.claude/skills/rai-quality-review/SKILL.md` | Modify — genericize all 4 steps |

## Acceptance Criteria

- **MUST:** QR detects project language from manifest or file extensions
- **MUST:** Python checks preserved (type:ignore, cast(), except, __all__)
- **MUST:** TypeScript checks added (as casts, any, unhandled promises)
- **MUST:** C# checks added (null safety, unchecked casts, IDisposable)
- **MUST:** 7 test heuristics remain universal
- **SHOULD:** Fallback to extension scanning when no manifest
- **MUST NOT:** Break existing Python QR behavior
