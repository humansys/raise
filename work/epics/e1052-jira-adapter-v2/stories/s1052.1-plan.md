## Data Flow Trace

```
JiraClient                    atlassian.Jira              Jira Cloud API
───────────                   ──────────────              ──────────────
get_issue(key)  ──────────►  issue(key)  ────────────►  GET /rest/api/3/issue/{key}
                                                          ◄── 200 | 401 | 404
                ◄──────────  dict | ApiError
                                    │
                              _map_error()
                                    │
                              ▼
                JiraNotFoundError | JiraAuthError | JiraApiError

from_config(config, instance)
    │
    ├── resolve instance from config.instances[instance]
    ├── _resolve_token(instance) → env: JIRA_API_TOKEN_{INST} > JIRA_API_TOKEN
    ├── _resolve_username(instance) → config.email > env: JIRA_USERNAME_{INST} > JIRA_USERNAME
    └── JiraClient(url, username, token)
            │
            └── atlassian.Jira(url, username, password, cloud=True, backoff_and_retry=True)
```

### Exception Mapping

```
atlassian.errors              jira_exceptions
────────────────              ───────────────
ApiPermissionError  ────────► JiraAuthError
ApiNotFoundError    ────────► JiraNotFoundError
ApiError            ────────► JiraApiError (with context)
Exception           ────────► JiraApiError (unexpected)
```
