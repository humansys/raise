# S338.11: Install and Configure Semgrep MCP Server

## User Story

**As a** framework developer,
**I want** Semgrep MCP server installed and configured in the project,
**so that** SAST, SCA, and secret detection are available via `rai mcp call`.

## Acceptance Criteria

```gherkin
Scenario: Semgrep MCP server is registered
  Given the project has .raise/mcp/ registry
  When I run `rai mcp list`
  Then I see semgrep listed as a registered server

Scenario: Semgrep health check passes
  Given semgrep MCP server is registered
  And SEMGREP_APP_TOKEN is configured
  When I run `rai mcp health semgrep`
  Then the server reports healthy

Scenario: Security scan works E2E
  Given semgrep MCP server is healthy
  When I run `rai mcp call semgrep <scan_tool> --args '{...}'`
  Then I receive scan results
```

## Size

XS — install, configure, validate
