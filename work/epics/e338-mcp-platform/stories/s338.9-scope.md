# Scope: S338.9 — Stack-aware MCP recommendations

## In Scope

- Stack-to-MCP mapping data (language/framework → recommended servers)
- Recommendation logic in a skill that checks detected stack vs registered servers
- Integration point: after `/rai-discover-scan` or via `/rai-mcp-status`

## Out of Scope

- Auto-install (recommendations only, human decides via `/rai-mcp-add`)
- Community MCP catalog / registry lookup
- Framework-level detection (e.g. React vs Vue) — language-level only for now

## Done When

- Rai suggests relevant MCP servers based on detected languages
- Already-registered servers are excluded from suggestions
- Suggestions include how to add via `/rai-mcp-add`
