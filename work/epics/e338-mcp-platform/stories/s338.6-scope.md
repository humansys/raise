# Scope: S338.6 — `rai mcp install`

## In Scope

- `rai mcp install <package> --type uvx|npx|pip --name NAME [--env ENV] [--force]`
- Type-specific command/args generation:
  - npx: `command: npx, args: ["-y", "<package>"]`
  - uvx: `command: uvx, args: ["<package>"]`
  - pip: `pip install <package>` then `command: python, args: ["-m", "<module>"]`
- Health check after install (connect + list_tools)
- Generate `.raise/mcp/<name>.yaml` via scaffold logic (reuse)
- Warning on health failure (non-fatal — install still succeeds)

## Out of Scope

- Auto-detecting --type from package name (AR-R3)
- npm/yarn/bun package managers
- Version pinning in config
- Uninstall command

## Done Criteria

- [ ] `rai mcp install` works for uvx, npx, pip types
- [ ] Health check runs after install, warns on failure
- [ ] Config YAML generated and parseable by McpServerConfig
- [ ] Overwrite protection (--force)
- [ ] pyright + ruff + pytest green
