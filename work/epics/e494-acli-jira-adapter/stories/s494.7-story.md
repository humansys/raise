# S494.7: ACLI Jira Adapter Documentation

## User Story

**As a** RaiSE developer or adopter,
**I want** clear documentation covering installation, configuration, architecture, and usage of the ACLI Jira adapter,
**So that** I can set up, understand, and contribute to the Jira integration without reading source code.

## Acceptance Criteria

```gherkin
Scenario: New developer sets up Jira adapter from docs
  Given a developer has never used the ACLI adapter
  When they follow the installation and configuration guide
  Then they can run rai backlog search successfully

Scenario: Developer understands adapter architecture
  Given a developer needs to modify the adapter
  When they read the architecture documentation
  Then they understand the component relationships and data flow
  And UML class and sequence diagrams are provided

Scenario: Multi-instance configuration is documented
  Given a developer needs to connect multiple Jira sites
  When they read the configuration reference
  Then they can set up jira.yaml with named instances and project routing
```
