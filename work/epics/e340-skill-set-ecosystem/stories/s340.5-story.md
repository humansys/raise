## What

Create `/rai-skillset-manage` — a conversational skill that guides teams
through skill set management using `rai skill set` CLI commands.

## Why

The CLI provides observable operations. The skill provides the guided
conversational UX on top — detecting context, asking the right questions,
and delegating to CLI for execution.

## Acceptance Criteria

- [ ] SKILL.md created in `.claude/skills/rai-skillset-manage/`
- [ ] Uses `rai skill set list/create/diff` commands
- [ ] Conversational flow: detect sets → choose action → execute via CLI
- [ ] ADR-040 compliant (≤150 lines, 7 sections, no TODOs)
- [ ] `rai skill validate` passes
