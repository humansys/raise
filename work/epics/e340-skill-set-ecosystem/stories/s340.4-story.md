## What

Create `rai skill set create|list|diff` CLI commands for observable
skill set management. Teams use these to create sets from builtins,
list existing sets, and compare sets against builtins.

## Why

Skill set management must go through CLI for observable workflow —
not embedded in a conversational skill. The CLI commands become the
foundation for `/rai-skillset-manage`.

## Acceptance Criteria

- [ ] `rai skill set create <name>` creates `.raise/skills/<name>/` from builtins
- [ ] `rai skill set create <name> --empty` creates empty set directory
- [ ] `rai skill set list` shows all sets in `.raise/skills/`
- [ ] `rai skill set diff <name>` shows which skills differ from builtins
- [ ] All commands have `--format human|json` output
- [ ] Tests cover each command
