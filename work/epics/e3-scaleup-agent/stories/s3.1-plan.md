### T2: Schema validator — 3 checks (M)

**Description:** Implement `validate_schema()` with 3 checks: competency question traversal, corpus coverage (H1 headings → node types), schema consistency (references + orphans). Reads YAML nodes from a configurable `knowledge_dir` path.

**Files:**
- `src/rai_agent/scaleup/validation/schema_validator.py` (create)
- `tests/scaleup/validation/test_schema_validator.py` (create)
- `tests/scaleup/validation/fixtures/` (create — minimal test fixtures: 3-4 YAML nodes, small corpus excerpt)

**TDD:**
- RED: Test each check independently with fixtures:
  - `test_competency_check_covered` — CQ with matching traversal path → covered
  - `test_competency_check_uncovered` — CQ with no matching path → uncovered with reason
  - `test_corpus_coverage` — H1 headings mapped to node types, gaps reported
  - `test_consistency_valid` — all references exist, no orphans → valid
  - `test_consistency_invalid_ref` — broken reference → reported
  - `test_consistency_orphan` — node with zero relationships → flagged
  - `test_validate_schema_full` — all 3 checks combined → `ValidationReport`
- GREEN: Implement validator
- REFACTOR: Extract helpers if needed

**Verification:**
```bash

uv run ruff check src/ tests/
uv run pyright
```

**Dependencies:** T1

## Execution Order

```
T1 (models + CQs) → T2 (validator) → T3 (integration)
```

Linear — each task builds on the previous. No parallelism needed for 3 tasks.

**Risk-first:** T2 is the riskiest (YAML parsing, traversal logic), but needs T1's models. T3 validates against real data early, before building extraction pipeline on top.

## Risks

| Risk | L/I | Mitigation |
|------|:---:|------------|
| Eduardo's YAML has inconsistencies (missing fields, wrong types) | M/M | T3 catches this. Validator reports, doesn't crash |
| Corpus H1 headings don't map cleanly to node types | L/M | Keyword heuristic with fallback to "unmapped". Report gaps, don't fail |
| PyYAML dependency needed | L/L | Already transitive via pydantic/other deps. Add explicitly if needed |
