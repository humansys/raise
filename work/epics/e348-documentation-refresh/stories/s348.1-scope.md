# S348.1: Documentation Audit — Scope

## In Scope

- Systematic gap analysis per Diataxis type (tutorial, how-to, reference, explanation)
- Current vs needed state for 3 audiences: users, developers, AI agents
- Audit of all documentation artifacts:
  - docs/ Astro site content (en/es)
  - README.md
  - AGENTS.md
  - llms.txt (missing — note absence)
  - CLI reference completeness (compare `rai --help` tree vs documented commands)
  - Extension point documentation (adapters, MCP, skills, hooks)
  - Community files (CONTRIBUTING.md, CODE_OF_CONDUCT.md, CHANGELOG.md, LICENSE)
- Priority ranking for subsequent stories
- Research input: `work/research/e348-documentation-practices/report.md`

## Out of Scope

- Writing or updating any documentation content (that is S348.2-S348.6)
- Astro site structural changes
- Automated doc generation tooling
- Spanish translation assessment (follow-up)
