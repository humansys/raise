# S348.3 Design — llms.txt + AGENTS.md

## Overview

Two static files at project root. No code changes, no tests needed.

## llms.txt

Follows https://llmstxt.org/ spec:
- H1: project name
- Blockquote: short description
- H2 sections with markdown link lists to documentation URLs

Sections: Docs (getting started + CLI), Concepts, Guides, Source.

## AGENTS.md

Comprehensive agent instructions covering:
- Quick start (run /rai-session-start)
- Project structure (src/, packages/, .raise/, etc.)
- Branch model
- Key commands
- Development rules (TDD, types, Pydantic)
- Extension points
- Governance files

## Risks

None. Static documentation files only.
