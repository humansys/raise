## Context

During E325 work, long-running sessions hit Claude Code's context compaction, losing working state. This spike explored an append-only JSONL journal as incremental memory that survives compaction, enabling the agent to restore context without human prompting.

## In Scope

- JournalEntry Pydantic model (decisions, completed tasks, blockers)
- Append-only JSONL writer with atomic writes
- CLI commands: `rai session journal add` and `rai session journal show`
- PreCompact hook for logging journal state before compaction
- Post-compaction restoration strategy (instruction-based via CLAUDE.md)
- Bug fixes discovered during spike (.envrc stdout pollution, hook JSON format)

## Out of Scope

- Hook-based post-compaction injection (confirmed broken — Claude Code bugs #12671, #15174, #16538)
- SessionStart hook restoration (pivot away during spike)
- Journal pruning or rotation
- Server-side journal persistence

## Done Criteria

- [x] JournalEntry model with typed fields and validation
- [x] Writer appends JSONL atomically, creates file on first write
- [x] CLI `add` creates entries, `show` reads with `--compact` format
- [x] PreCompact hook logs journal state (side-effect only)
- [x] CLAUDE.md contains post-compaction restoration instructions
- [x] .envrc interactive guard prevents stdout pollution
- [x] 10 tests pass
