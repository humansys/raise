# Scope: S325.2 — Delegation Profile Model

**Epic:** E325 Agent-Orchestrated Workflow
**Jira:** RAISE-331
**Size:** S

- Add `DelegationLevel` enum (review/notify/auto) to profile module
- Add `DelegationConfig` Pydantic model (default_level + overrides dict)
- Add `delegation` field to `DeveloperProfile`
- Add `resolve_delegation(profile, skill_name)` function — returns effective level
- ShuHaRi derivation: shu→review, ha→notify, ri→auto

- Delegation auto-tuning from history
- CLI commands for delegation management
