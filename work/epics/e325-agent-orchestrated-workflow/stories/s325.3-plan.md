### T2: Add `agent` format to graph commands (S)

**Files:** `src/rai_cli/cli/commands/graph.py`, `tests/cli/commands/test_graph.py`

**TDD Cycle:**
- RED: Test `query --format agent` produces `type|id|content` lines, test `context --format agent` produces expected structure, test `list --format agent` produces `type|count` summary
- GREEN: Add `_format_agent()` for query, `_format_context_agent()` for context, `_format_concepts_agent()` for list. Wire into existing format routing.

**Verification:** `pytest tests/cli/commands/test_graph.py -v && ruff check src/rai_cli/cli/commands/graph.py && pyright src/rai_cli/cli/commands/graph.py`

**AC:** MUST-1 (graph query, context, list), MUST-2 (human unchanged), MUST-3 (no truncation), MUST-4 (list as summary), MUST-5 (error pattern), SHOULD-1 (compact compat)
