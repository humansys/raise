# S1051.6: Config Generator — `/rai-adapter-setup` Confluence

## User Story

**As a** RaiSE developer setting up Confluence integration for the first time,
**I want** an interactive setup skill that discovers my Confluence instance and generates a valid config,
**so that** I don't have to hand-write YAML or guess space keys, page IDs, or routing rules.

## Acceptance Criteria (Gherkin)

```gherkin
Scenario: Discovery finds available spaces
  Given a valid Confluence API token in the environment
  When I run the config generator
  Then I see a list of available spaces with keys and names
  And I can select which space to use

Scenario: Generator produces valid YAML
  Given I have selected a space and confirmed routing options
  When the generator writes confluence.yaml
  Then the output passes Pydantic validation (ConfluenceConfig model)
  And the file is written to .raise/confluence.yaml

Scenario: Existing config detected
  Given .raise/confluence.yaml already exists
  When I run the config generator
  Then I am warned about the existing config
  And I can choose to overwrite or abort

Scenario: Discovery fails gracefully
  Given the Confluence API is unreachable
  When I run the config generator
  Then I see a clear error message with troubleshooting hints
  And the generator does not write a partial config

Scenario: Pure logic is testable without interaction
  Given the discovery results and user selections as data
  When I call the config builder function
  Then it returns a valid ConfluenceConfig model
  And no I/O or prompting occurs
```

## Specification by Example

```python
# Pure logic: build config from selections
from raise_cli.adapters.confluence.config import ConfluenceConfig

config = build_confluence_config(
    url="https://mysite.atlassian.net/wiki",
    username="user@example.com",
    space_key="RaiSE1",
    routing={
        "adr": {"parent_title": "Architecture Decision Records"},
        "roadmap": {"parent_title": "Roadmaps"},
    },
)
assert isinstance(config, ConfluenceConfig)
assert config.instances["default"].space_key == "RaiSE1"

# YAML output matches schema
import yaml
yaml_str = config_to_yaml(config)
parsed = yaml.safe_load(yaml_str)
assert "instances" in parsed
assert "routing" in parsed["instances"]["default"]
```
