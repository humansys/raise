# S1051.5: Adapter Doctor — Confluence

## User Story

**As a** RaiSE developer using Confluence integration,
**I want** `rai doctor` to validate my Confluence config against the live backend,
**so that** I get actionable feedback when config references non-existent spaces, pages, or labels.

## Acceptance Criteria (Gherkin)

```gherkin
Scenario: Config references valid space
  Given a confluence.yaml with space_key "RaiSE1"
  And the Confluence instance has space "RaiSE1"
  When I run rai doctor --online
  Then I see a PASS result for "confluence-space-exists"

Scenario: Config references non-existent space
  Given a confluence.yaml with space_key "MISSING"
  And the Confluence instance does not have space "MISSING"
  When I run rai doctor --online
  Then I see an ERROR result for "confluence-space-exists"
  And the fix_hint suggests checking the space key

Scenario: Routing parent page exists
  Given a routing rule with parent_title "Architecture Decision Records"
  And that page exists in the configured space
  When I run rai doctor --online
  Then I see a PASS result for "confluence-routing-parent"

Scenario: Routing parent page missing
  Given a routing rule with parent_title "Nonexistent Page"
  And that page does not exist in the configured space
  When I run rai doctor --online
  Then I see an ERROR result for "confluence-routing-parent"
  And the fix_hint includes the missing page title

Scenario: Discovery service unavailable
  Given the Confluence API is unreachable
  When I run rai doctor --online
  Then the check reports ERROR with connectivity hint
  And the check does not crash rai doctor

Scenario: Offline mode skips live validation
  Given a valid confluence.yaml
  When I run rai doctor (without --online)
  Then config-vs-backend checks are skipped
  And only config file and env var checks run
```

## Specification by Example

```python
# Expected CheckResult for space mismatch:
CheckResult(
    check_id="adapter-confluence-space-exists",
    category="adapters",
    status=CheckStatus.ERROR,
    message="Confluence space 'MISSING' not found on instance",
    fix_hint="Check space_key in .raise/confluence.yaml — available spaces: RaiSE1, DEMO",
)

# Expected CheckResult for missing routing parent:
CheckResult(
    check_id="adapter-confluence-routing-parent",
    category="adapters",
    status=CheckStatus.ERROR,
    message="Routing parent page 'Nonexistent Page' not found in space 'RaiSE1'",
    fix_hint="Create page 'Nonexistent Page' in space RaiSE1, or update routing in confluence.yaml",
)
```
