# S1051.7: FilesystemDocsTarget + CompositeDocTarget — Design

## Problem
Skills need a single `rai docs publish` path that writes to both Confluence and
local filesystem. Current resolver errors on 2+ targets. No filesystem docs target exists.

## Value
Single publish path for skills. Local files for git/offline. Confluence for shared docs.
Enables S1051.8/S1051.9 skill migration.

## Complexity: Moderate
3 new components (FS target, composite, resolver change), clear contracts.

## Approach

### Decisions

| ID | Decision | Rationale |
|----|----------|-----------|
| D1 | FilesystemDocsTarget is write-focused | search/get_page not meaningful on filesystem — raise NotImplementedError |
| D2 | CompositeDocTarget: primary = first target | health/get/search delegate to first; publish to all |
| D3 | Resolver: 2+ targets → auto-compose | No flag needed for common case; `--target X` for single |
| D4 | File path from metadata["path"] | Skills control where files land — adapter just writes |
| D5 | Filesystem always first in composite | Local write succeeds even if Confluence is down |
| D6 | publish returns primary (Confluence) result | User cares about remote success; local is best-effort |

### FilesystemDocsTarget

```python
class FilesystemDocsTarget:
    """Writes documentation artifacts as local markdown files."""

    def __init__(self, project_root: Path | None = None) -> None:
        self._root = project_root or Path.cwd()

    def can_publish(self, doc_type: str, metadata: dict[str, Any]) -> bool:
        return "path" in metadata  # needs a file path

    def publish(self, doc_type: str, content: str, metadata: dict[str, Any]) -> PublishResult:
        path = self._root / metadata["path"]
        path.parent.mkdir(parents=True, exist_ok=True)
        path.write_text(content)
        return PublishResult(success=True, url=str(path), message="Written to filesystem")

    def get_page(self, identifier: str) -> PageContent:
        raise NotImplementedError("FilesystemDocsTarget does not support get_page")

    def search(self, query: str, limit: int = 10) -> list[PageSummary]:
        return []  # no search on filesystem

    def health(self) -> AdapterHealth:
        return AdapterHealth(name="filesystem-docs", healthy=True)
```

### CompositeDocTarget

```python
class CompositeDocTarget:
    """Delegates to multiple DocumentationTargets. Primary is first in list."""

    def __init__(self, targets: list[Any]) -> None:  # Any to avoid Protocol import
        self._targets = targets
        self._primary = targets[0]

    def can_publish(self, doc_type: str, metadata: dict[str, Any]) -> bool:
        return any(t.can_publish(doc_type, metadata) for t in self._targets)

    def publish(self, doc_type: str, content: str, metadata: dict[str, Any]) -> PublishResult:
        primary_result: PublishResult | None = None
        for target in self._targets:
            if target.can_publish(doc_type, metadata):
                result = target.publish(doc_type, content, metadata)
                if primary_result is None:
                    primary_result = result
        return primary_result or PublishResult(success=False, message="No target accepted")

    def get_page(self, identifier: str) -> PageContent:
        return self._primary.get_page(identifier)

    def search(self, query: str, limit: int = 10) -> list[PageSummary]:
        return self._primary.search(query, limit=limit)

    def health(self) -> AdapterHealth:
        return self._primary.health()
```
