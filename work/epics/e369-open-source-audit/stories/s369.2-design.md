# S369.2 -- Package Metadata & Deps: Design

## Problem

PyPI listing lacks project URLs (Homepage, Docs, Repository, Changelog),
contains a dead `tomli` dependency (guarded for python < 3.11 but project
requires >= 3.12), and is missing the Python 3.13 classifier.

## Approach

Direct edits to pyproject.toml. Run pip-audit for vulnerability scan.

## Changes

| File | What to fix |
|------|-------------|
| pyproject.toml | Add `[project.urls]` with Homepage, Documentation, Repository, Changelog |
| pyproject.toml | Remove dead `tomli` dependency |
| pyproject.toml | Add `Programming Language :: Python :: 3.13` classifier |

## Verification

```bash
# Confirm no runtime tomli usage
grep -r "import tomli" src/

# Confirm URLs present
grep -A4 "\[project.urls\]" pyproject.toml

# Vulnerability scan
pip-audit
```

## Decisions

- **D-1:** Direct pyproject.toml edits, no migration tooling. Single file,
  3 small changes.
- **D-2:** Grep for tomli imports before removal to confirm no runtime usage.
