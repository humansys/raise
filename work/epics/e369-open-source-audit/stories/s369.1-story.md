# S369.1 -- Documentation Polish

## User Story

As a developer evaluating raise-commons,
I want all community docs to have accurate names, URLs, and version references,
so that my first impression reflects a well-maintained project.

## Acceptance Criteria

- CONTRIBUTING.md references `raise-cli` and `src/raise_cli/` (not old `rai-cli` / `src/rai_cli/`)
- SECURITY.md supported versions table reflects current 2.x releases (not stale "2.0.0-alpha.x")
- CHANGELOG.md compare URLs point to `humansys-ai/raise-commons` (not `humansys/raise`)
- CHANGELOG.md has proper blank lines between version entries
- README has a CI badge
- README Python 3.14 claim is verified and corrected if needed

## Size

XS -- text edits only, no code changes.
