# S463.3: Rename raise-cli

## User Story

As a developer installing the RaiSE CLI,
I want the package to be `raise-cli` with namespace `raise_cli`,
So that `pip install raise-cli` resolves all dependencies correctly without
namespace collisions.

## Acceptance Criteria

Given the rename is applied
When I run `uv run rai --version`
Then the CLI responds (command unchanged)

Given all imports are updated
When I run `grep -r 'rai_cli' src/ tests/ --include='*.py'`
Then zero results (excluding __pycache__)

Given all mock paths are updated
When I run `uv run pytest`
Then 3699+ tests pass
