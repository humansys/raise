# Plan: S347.6 ‚Äî `rai backlog sync` mirror generation

## Overview

Implement `rai backlog sync` to regenerate `governance/backlog.md` from a
remote adapter (Jira). Core sync logic lives in `src/rai_cli/backlog/sync.py`;
the CLI subcommand is wired into the existing `backlog_app` Typer group.

**Risk-first ordering:** T1 tackles the core sync logic (highest risk ‚Äî new
module, Pydantic model, atomic write, formatting). T2 adds filesystem
detection (simple but critical guard). T3 wires CLI (integration point). T4
covers adapter error handling. T5 is manual integration test.

## Tasks

### T1: Core sync logic ‚Äî `sync_backlog()` + `SyncResult` model

**Description:** Create `src/rai_cli/backlog/sync.py` with:
- `SyncResult` Pydantic model (adapter_name, epic_count, timestamp, output_path)
- `sync_backlog(adapter, adapter_name, project_filter, output_path) -> SyncResult`
- `_format_markdown(epics, adapter_name, timestamp) -> str` ‚Äî generates markdown table
- `_atomic_write(path, content) -> None` ‚Äî temp file + rename

**Files:**

- Create: `src/rai_cli/backlog/sync.py`

### T2: Filesystem adapter detection ‚Äî skip self-sync

**Description:** Add test and logic for `FilesystemPMAdapter` isinstance check.
When the resolved adapter is `FilesystemPMAdapter`, `sync_backlog` should raise
a specific exception (or return a sentinel) that the CLI can catch to print a
no-op message.

Design decision: `sync_backlog` raises `ValueError("Filesystem adapter is source
of truth")` when passed a `FilesystemPMAdapter`. The CLI catches this and prints
the message + exits cleanly.

**Files:**
- Modify: `tests/backlog/test_sync.py`
- Modify: `src/rai_cli/backlog/sync.py`

**TDD cycle:**
- RED: Test that passing a `FilesystemPMAdapter` to `sync_backlog` raises
  `ValueError` with expected message.
- GREEN: Add isinstance check at top of `sync_backlog`.
- REFACTOR: Verify import is clean, no circular dependency.

**Verification:**
```bash
uv run pytest tests/backlog/test_sync.py -v -k filesystem
uv run pyright src/rai_cli/backlog/sync.py
```

**Dependencies:** T1

## Risks & Mitigations

| Risk | Impact | Mitigation |
|------|--------|------------|
| `IssueSummary` lacks fields for table | Medium | Design already maps key/summary/status/issue_type ‚Äî all present in `IssueSummary` |

| Adapter `search()` returns different shapes | Low | `IssueSummary` is the protocol return type ‚Äî all adapters conform |
| `backlog/` package doesn't exist yet | Low | Create `__init__.py` in T1 |
