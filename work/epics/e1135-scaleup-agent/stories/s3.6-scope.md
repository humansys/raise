# S3.6 Scope — Knowledge Query CLI

## In Scope

- `rai knowledge query <domain> <query>` CLI command
- Load domain graph from YAML nodes via ScaleUpGraphBuilder
- Run retrieve() with domain's DomainAdapter
- Human-readable and JSON output formats
- Domain auto-detection when only one domain exists
- Top-K parameter (default 10)

## Out of Scope

- Persistent graph index (build on each query is acceptable for S size)
- LLM-generated responses from retrieved nodes (future epic: domain packs)
- Dynamic adapter registration in domain.yaml (future epic)
- Caching or performance optimization
- New domains beyond ScaleUp

## Done When

- `rai knowledge query scaleup "cash flow"` returns ranked nodes from 447 extracted
- `rai knowledge query scaleup "leadership team" --json` returns machine-readable output
- Integration test validates full chain: YAML → graph → retrieve → formatted output
- CLI registered in knowledge app (existing `knowledge/cli.py`)
