# S3.8 Scope — Full-Cycle Extraction from Zero

## In Scope

- Schema discovery: analyze corpus → propose node types + fields (using existing extraction pipeline from S3.2)
- Full extraction run: corpus → YAML nodes using discovered schema
- Gate validation: run S3.7 gates on extracted output (validate, reconcile, coverage, graph)
- Diff report: compare extracted nodes vs Eduardo's manually curated nodes
- Document epistemological findings (what the machine finds vs what the expert curated)

## Out of Scope

- New extraction algorithms (reuse S3.2 pipeline)
- Modifying Eduardo's curated nodes
- UI or interactive discovery workflow
- Multi-corpus support (single corpus: ScaleUp book)
- Automatic schema refinement loops

## Done Criteria

1. Full extraction cycle runs end-to-end without manual intervention (after initial config)
2. S3.7 gates produce results on extracted output (pass or actionable failures)
3. Diff report exists comparing machine vs manual nodes (count + overlap analysis)
4. Findings documented for retrospective
