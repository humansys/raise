## Data Flow Trace (PAT-E-001)

```
corpus_path ‚Üí discover_schema(LLM) ‚Üí SchemaSpec(node_types[], relationship_types[])
SchemaSpec + OntologyNode.model_fields ‚Üí diff_schemas() ‚Üí SchemaDiffReport(types_both, types_only_*, field_diffs)
corpus_path + examples_dir=None ‚Üí ExtractionPipeline.run() ‚Üí ExtractionResult(nodes[], reconciliation, coverage)
ExtractionResult.nodes ‚Üí write_yaml() ‚Üí .raise/knowledge/scaleup/extracted/*.yaml
extracted_dir + curated_dir ‚Üí diff_nodes() ‚Üí NodeDiffReport(nodes_both, nodes_only_*, overlap_pct, by_decision)
extracted/*.yaml ‚Üí rai knowledge check scaleup ‚Üí GateResult[] (validate, reconcile, coverage, graph)
```
