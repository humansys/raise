## Risks

| Risk | Mitigation |
|------|-----------|
| `FilesystemAdapter.append()` reads full file then rewrites — performance on large JSONL | Acceptable: largest file (patterns.jsonl) is ~100 lines. Flagged for future if files grow. |
| `last_modified` field missing in existing YAML files | Default `None` — timestamp check skipped when either side is None |
| `rel_path` computation in `save_session_state` may fail if `state_path` not under `project_path` | Session state path is always `.raise/rai/session-state.yaml` under project — safe |
