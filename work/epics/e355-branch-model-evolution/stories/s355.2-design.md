# S355.2 Design — Clean up stale epic-branch references

## Complexity: Simple
Documentation-only changes. Per PAT-E-445: blast radius grep IS the design.

## Problem
S355.1 updated the branch model in CLAUDE.md and core skills, but several live files still reference the old 3-level model (dev → epic → story). This creates contradictions that confuse both humans and AI agents.

## Value
All live documentation consistently reflects the actual 2-level branch model, eliminating workflow confusion.

## Approach
Grep-driven find-and-replace across identified files. No code changes.
