# S1286.3: Build /rai-bugfix-run orchestrator

## Objective

Create a `/rai-bugfix-run` orchestrator that runs the full 7-phase bugfix pipeline via subagent fork isolation, with a mandatory anti-anchoring triage gate and delegation-based gates for analyse and fix.

## In Scope

- `/rai-bugfix-run` SKILL.md following `/rai-story-run` fork pattern
- Phase detection from `work/bugs/RAISE-{N}/` artifacts for resume
- Mandatory HITL triage gate with anti-anchoring presentation (D3)
- Delegation gates after analyse and fix (REVIEW/NOTIFY/AUTO)
- All 7 phases fork via Agent tool with fresh context
- Completion report with artifact table

## Out of Scope

- Modifying the 7 atomic skills (S1286.2 — done)
- Pipeline engine integration (E1065, 3.0.0)
- Dogfood validation (S1286.4)

## Done When

- [ ] `/rai-bugfix-run RAISE-{N}` orchestrates all 7 phases with subagent fork isolation
- [ ] Triage gate is mandatory with active verification (not approve/reject)
- [ ] Phase detection resumes from most advanced artifact
- [ ] Delegation gates work for analyse and fix
- [ ] Close agent prompt includes scope constraints
- [ ] Skill validates via `rai skill validate`
