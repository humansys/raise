# S1286.2: Extract 7 atomic bugfix skills

## Objective

Extract each of the 7 Steps from the monolithic `/rai-bugfix` skill (290 lines, ~5000 words) into individual SKILL.md files, each ~500-700 words, within the AgentIF high-compliance zone.

## In Scope

- 7 SKILL.md files in `.claude/skills/rai-bugfix-{phase}/`
- Skill names matching E1065 `bugfix.yaml`: start, triage, analyse, plan, fix, review, close
- Verbatim extraction of step content (D1: extract, don't rewrite)
- Focused Context section per skill (inputs, expected state, when to use)
- Each skill independently invocable (not just through orchestrator)
- Skillset distribution copy in `.raise/skills/raise-maintainability/rai-bugfix-{phase}/`

## Out of Scope

- Orchestrator (`/rai-bugfix-run`) — S1286.3
- Modifying the monolithic `/rai-bugfix` — preserved as fallback
- Overlays (raise-dev, raise-dev-ts) — only apply to fix phase, deferred
