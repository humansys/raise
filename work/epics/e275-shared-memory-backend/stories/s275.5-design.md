# S275.5 — Agent telemetry + memory endpoints — Design

## Problem

Rovo agents execute governance skills but have no way to report what they did or feed learnings back to Rai's memory. Without telemetry, we can't track skill execution or measure POC success. Without memory endpoints, patterns discovered by Rovo die with the conversation.

## Value

Fernando gets a complete OpenAPI spec (graph + telemetry + memory) to build Forge tools. We get observability into Rovo's governance actions and a feedback loop for Rai's shared memory.

## Approach

Two new domains, same three-layer pattern as S275.4. Both are append-only for the POC (no update/delete).

**DB:** One table `agent_events` (telemetry) + one table `memory_patterns` (learned patterns). Both per-org, JSONB payload for flexibility.

**Routers:** `api/v1/agent.py` (events) + `api/v1/memory.py` (patterns). Separate routers for clean OpenAPI grouping.

### Components

| Component | Change | File |
|-----------|--------|------|
| DB models: `AgentEventRow`, `MemoryPatternRow` | CREATE | `db/models.py` (append) |

| Routes: memory router | CREATE | `api/v1/memory.py` |
| App factory: register routers | MODIFY | `app.py` |
| Tests (4 layers × 2 domains) | CREATE | `tests/rai_server/test_*` |

### Missing-Entity Strategy (PAT-E-523)

Not applicable — both domains are append-only. No GET-by-id needed for POC. Lists return empty array if no data.

## Examples

### DB Models

```python
class AgentEventRow(Base):
    __tablename__ = "agent_events"

    id: Mapped[uuid.UUID] = mapped_column(Uuid, primary_key=True, server_default=func.gen_random_uuid())
    org_id: Mapped[uuid.UUID] = mapped_column(ForeignKey("organizations.id"), index=True)
    event_type: Mapped[str] = mapped_column(String(100), index=True)
    payload: Mapped[dict] = mapped_column(JSONB, server_default="{}")
    created_at: Mapped[datetime] = mapped_column(DateTime(timezone=True), server_default=func.now())


class MemoryPatternRow(Base):
    __tablename__ = "memory_patterns"

    id: Mapped[uuid.UUID] = mapped_column(Uuid, primary_key=True, server_default=func.gen_random_uuid())
    org_id: Mapped[uuid.UUID] = mapped_column(ForeignKey("organizations.id"), index=True)
    content: Mapped[str] = mapped_column(Text)
    context: Mapped[list] = mapped_column(JSONB, server_default="[]")  # keyword tags
    properties: Mapped[dict] = mapped_column(JSONB, server_default="{}")
    created_at: Mapped[datetime] = mapped_column(DateTime(timezone=True), server_default=func.now())
```

### Schemas

```python
# schemas/agent.py
class AgentEventCreate(BaseModel):
    event_type: str = Field(min_length=1, max_length=100)
    payload: dict[str, Any] = Field(default_factory=dict)

class AgentEventResponse(BaseModel):
    id: uuid.UUID

    id: uuid.UUID
    event_type: str
    payload: dict[str, Any]
    created_at: datetime

class AgentEventListResponse(BaseModel):
    events: list[AgentEventItem]
    total: int

# schemas/memory.py
class MemoryPatternCreate(BaseModel):
    content: str = Field(min_length=1)
    context: list[str] = Field(default_factory=list)
    properties: dict[str, Any] = Field(default_factory=dict)

    id: uuid.UUID
    status: str = "ok"

class MemoryPatternItem(BaseModel):
    id: uuid.UUID
    content: str
    context: list[str]
    properties: dict[str, Any]
    created_at: datetime

class MemoryPatternListResponse(BaseModel):
    patterns: list[MemoryPatternItem]
    total: int
```

### Service Layer

```python
# services/agent.py
async def record_event(session_factory, org_id, body: AgentEventCreate):
    async with session_factory() as session, session.begin():
        event_id = await insert_event(session, org_id, body.event_type, body.payload)
    return AgentEventResponse(id=event_id)

async def get_events(session_factory, org_id, limit):
    async with session_factory() as session:
        rows = await list_events(session, org_id, limit)
    return AgentEventListResponse(events=[AgentEventItem(**r) for r in rows], total=len(rows))
```

```python
# services/memory.py
```

### DB Queries

```python
# db/agent_queries.py
async def insert_event(session, org_id, event_type, payload) -> uuid.UUID:
    stmt = sa_insert(AgentEventRow).values(org_id=org_id, event_type=event_type, payload=payload).returning(AgentEventRow.id)
    result = await session.execute(stmt)
    return result.scalar_one()

async def list_events(session, org_id, limit) -> list[dict]:
    stmt = select(AgentEventRow).where(AgentEventRow.org_id == org_id).order_by(AgentEventRow.created_at.desc()).limit(limit)
    result = await session.execute(stmt)
    return [row_to_dict(r) for r in result.scalars()]
```

## Acceptance Criteria

### MUST
1. `POST /api/v1/agent/events` stores event, returns id
2. `GET /api/v1/agent/events` lists events newest-first with limit param
3. `POST /api/v1/memory/patterns` stores pattern, returns id
4. `GET /api/v1/memory/patterns` lists patterns for org
5. Auth required on all endpoints (OrgContext)
6. All tests pass at 4 layers (schemas, db, services, API)

### SHOULD
1. JSONB payload/properties for flexible schema (iterate with Fernando)
2. `context` field as keyword list for future filtering

### MUST NOT
1. No event analytics / aggregation
2. No pattern reinforcement / scoring (future)
3. No update/delete endpoints (append-only for POC)
