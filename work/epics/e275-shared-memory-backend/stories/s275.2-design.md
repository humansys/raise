# S275.2: Design — PostgreSQL Schema + Alembic

**Complexity:** Moderate (4 components, schema decisions affect full epic)

## Problem

rai-server needs a persistence layer before any API endpoint can be built. Without DB models and migrations, S275.3–S275.7 are blocked.

## Value

Establishes the data foundation for the entire shared memory backend. Schema decisions made here propagate to every subsequent story.

## Approach

Create `rai-server` as a uv workspace package with SQLAlchemy 2.0 async models, Alembic migrations, and Docker Compose for local dev.

### Components

| Component | Change | Description |
|-----------|--------|-------------|
| `packages/rai-server/` | CREATE | New uv workspace package |
| `packages/rai-server/src/rai_server/db/models.py` | CREATE | 4 SQLAlchemy models |
| `packages/rai-server/src/rai_server/db/session.py` | CREATE | Async engine + session factory |
| `packages/rai-server/alembic/` | CREATE | Alembic config + initial migration |
| `docker-compose.yml` | CREATE | PG 16 + server placeholder |
| `packages/rai-server/pyproject.toml` | CREATE | Package metadata + deps |

### Design Decisions

| # | Decision | Choice | Rationale |
|---|----------|--------|-----------|
| 1 | Package name | `rai-server` | Generic, extensible, no naming debt |
| 2 | Internal structure | Flat `db/` module | Simple first, split when justified |
| 3 | SQLAlchemy style | `Mapped[]` columns (2.0) | Type-safe, pyright-friendly, "type everything" rule |
| 4 | UUID strategy | DB-side `gen_random_uuid()` | Built-in PG 13+, no extension needed |
| 5 | Domain mapping | Separate SA models, explicit mapping | Clean separation; `to_domain()` deferred to S275.4 |
| 6 | Docker scope | PG + server placeholder | Structure ready for S275.3 |
| 7 | Testing | Unit tests with SA inspection API | No real DB needed; integration tests in S275.3+ |

## Examples

### SQLAlchemy Models

```python
from sqlalchemy.orm import DeclarativeBase, Mapped, mapped_column, relationship
from sqlalchemy import String, DateTime, Float, ForeignKey, Text, Boolean, func
from sqlalchemy.dialects.postgresql import UUID, JSONB
import uuid
from datetime import datetime

class Base(DeclarativeBase):
    pass

class Organization(Base):
    __tablename__ = "organizations"

    id: Mapped[uuid.UUID] = mapped_column(
        UUID(as_uuid=True), primary_key=True, server_default=func.gen_random_uuid()
    )
    name: Mapped[str] = mapped_column(String(255))
    slug: Mapped[str] = mapped_column(String(63), unique=True, index=True)
    created_at: Mapped[datetime] = mapped_column(
        DateTime(timezone=True), server_default=func.now()
    )

class ApiKey(Base):
    __tablename__ = "api_keys"

    id: Mapped[uuid.UUID] = mapped_column(
        UUID(as_uuid=True), primary_key=True, server_default=func.gen_random_uuid()
    )
    org_id: Mapped[uuid.UUID] = mapped_column(ForeignKey("organizations.id"))
    key_hash: Mapped[str] = mapped_column(String(128))
    prefix: Mapped[str] = mapped_column(String(12))  # "rsk_abc123..."
    is_active: Mapped[bool] = mapped_column(Boolean, default=True)
    created_at: Mapped[datetime] = mapped_column(
        DateTime(timezone=True), server_default=func.now()
    )

class GraphNodeRow(Base):
    __tablename__ = "graph_nodes"

    id: Mapped[uuid.UUID] = mapped_column(
        UUID(as_uuid=True), primary_key=True, server_default=func.gen_random_uuid()
    )
    org_id: Mapped[uuid.UUID] = mapped_column(ForeignKey("organizations.id"), index=True)
    repo_id: Mapped[str] = mapped_column(String(255), index=True)
    scope: Mapped[str] = mapped_column(String(20))  # PROJECT | ORG
    node_type: Mapped[str] = mapped_column(String(50), index=True)
    node_id: Mapped[str] = mapped_column(String(255))  # domain ID (e.g. "PAT-001")
    content: Mapped[str] = mapped_column(Text)
    source_file: Mapped[str | None] = mapped_column(String(500), nullable=True)
    properties: Mapped[dict] = mapped_column(JSONB, default=dict)
    created_at: Mapped[datetime] = mapped_column(
        DateTime(timezone=True), server_default=func.now()
    )
    updated_at: Mapped[datetime] = mapped_column(
        DateTime(timezone=True), server_default=func.now(), onupdate=func.now()
    )

class GraphEdgeRow(Base):
    __tablename__ = "graph_edges"

    id: Mapped[uuid.UUID] = mapped_column(
        UUID(as_uuid=True), primary_key=True, server_default=func.gen_random_uuid()
    )
    org_id: Mapped[uuid.UUID] = mapped_column(ForeignKey("organizations.id"), index=True)
    repo_id: Mapped[str] = mapped_column(String(255))
    source_id: Mapped[uuid.UUID] = mapped_column(ForeignKey("graph_nodes.id"))
    target_id: Mapped[uuid.UUID] = mapped_column(ForeignKey("graph_nodes.id"))
    edge_type: Mapped[str] = mapped_column(String(50), index=True)
    weight: Mapped[float] = mapped_column(Float, default=1.0)
    properties: Mapped[dict] = mapped_column(JSONB, default=dict)
    created_at: Mapped[datetime] = mapped_column(
        DateTime(timezone=True), server_default=func.now()
    )
```

## Acceptance Criteria

**MUST:**
- `uv sync` resolves rai-server as workspace package with rai-core dependency
- 4 SQLAlchemy models with correct columns, types, FKs, and indexes
- Alembic initial migration creates all tables from clean DB
- `docker compose up -d` starts PG 16 on port 5432
- Unit tests pass for model structure (inspection-based)

**SHOULD:**
- GIN indexes on JSONB `properties` columns
- Composite unique constraint on `(org_id, repo_id, node_id)` for GraphNodeRow

**MUST NOT:**
- No FastAPI code (S275.3)
- No API endpoints (S275.4/S275.5)
- No `to_domain()` mapping methods yet (S275.4)
