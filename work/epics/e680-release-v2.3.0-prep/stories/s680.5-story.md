# S680.5: Smoke Test & Verification

## User Story

As a release engineer,
I want to verify that all v2.3.0 features work end-to-end beyond unit tests,
So that I can be confident the release is reliable for users.

## Context

S680.4 (quality gates & security) is done. This story runs end-to-end smoke tests
for all major v2.3.0 features before the release publish step (S680.6).

## Verifications

1. **Session roundtrip** (E654) -- session start/close from main repo
2. **ACLI Jira adapter integration** (E494) -- smoke test adapter operations
3. **Clean install test** -- fresh venv, pip install, basic commands
4. **`rai doctor` command** (RAISE-614) -- verify diagnostic output
5. **Pattern add default scope** (RAISE-608) -- verify scope defaults
6. **CLI extensions discovery** -- verify extension loading

## Output

Results report (`s680.5-results.md`) with pass/fail per verification and any issues found.
