# Story S493.2: Profile Export/Import — Scope

## In Scope

- `rai profile export [--output PATH]` CLI subcommand
- `rai profile import PATH [--force]` CLI subcommand
- Export logic: read profile, strip machine-local fields, add _meta
- Import logic: validate bundle, strip machine-local, save profile
- Confirmation prompt on import when profile already exists (--force to skip)
- Tests for export, import, validation, edge cases

## Out of Scope

- Exporting Claude Code memory (~/.claude/)
- Exporting MCP server configuration
- Encrypting the bundle
- Network transfer (user handles file copy)

## Done When

- `rai profile export` outputs valid YAML bundle
- `rai profile import` restores profile on new machine
- Machine-local fields (active_sessions, current_session) stripped
- Tests pass for all Gherkin scenarios
- Type checks and lint clean
