## Task 2: GitEvidenceExtractor — commits + merge commits

**Goal:** Create the extractor class with commit and merge commit extraction.

**Files:**
- CREATE `src/raise_cli/compliance/extractors/__init__.py`
- CREATE `src/raise_cli/compliance/extractors/git.py` — GitEvidenceExtractor class
- CREATE `tests/unit/compliance/extractors/__init__.py`
- CREATE `tests/unit/compliance/extractors/test_git.py`

**Implementation:**
- `GitEvidenceExtractor(repo_path: Path)` constructor
- `extract(mapping: ControlMapping, start_date: date, end_date: date) -> list[EvidenceItem]` public method
- `_extract_commits()` — `git log --after --before --format="%H|%an|%aI|%s"`
- `_extract_merge_commits()` — `git log --merges --after --before --format="%H|%an|%aI|%s"`
- Iterates mapping.controls, filters for `type: "git"` sources, dispatches by extractor name

**Tests:**
- Mock subprocess.run to return known git log output
- Verify correct EvidenceItem creation (control_id, evidence_type, fields)
- Date range filtering
- Empty results (no commits in range)
- Malformed git output handling (graceful skip)

**Verify:**
```bash
uv run pytest tests/unit/compliance/extractors/test_git.py -v
uv run ruff check src/raise_cli/compliance/extractors/
uv run pyright src/raise_cli/compliance/extractors/
```

**Commit:** `feat(s479.2): add git extractor for commits and merge commits`

## Task 3: GitEvidenceExtractor — tags + branches

**Goal:** Add tag and branch extraction to complete git evidence coverage.

**Files:**
- EDIT `src/raise_cli/compliance/extractors/git.py` — add `_extract_tags()`, `_extract_branches()`
- EDIT `tests/unit/compliance/extractors/test_git.py` — add tag/branch tests

**Implementation:**
- `_extract_tags()` — `git tag --sort=-creatordate --format="%(refname:short)|%(creatordate:iso-strict)|%(subject)"`, post-filter by date range
- `_extract_branches()` — `git branch -a --format="%(refname:short)|%(committerdate:iso-strict)"`

**Tests:**
- Tag extraction with date filtering (tags have no --after/--before, filter post-extraction)
- Branch extraction and naming pattern evidence
- Empty tag/branch results

**Verify:**
