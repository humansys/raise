# Private Content

This directory contains internal content not intended for public consumption.

## Structure

| Directory | Content | Purpose |
|-----------|---------|---------|
| `decisions/` | ADRs (Architecture Decision Records) | Framework development decisions |
| `business/` | Business model, market context, stakeholder map | Business stakeholder documents |
| `planning/` | Roadmaps, current state | Internal planning and status |
| `work-artifacts/` | Session logs, issue tracking, backlog | Development work artifacts |
| `tools/` | Ontology prompts, validation tools, cursor rules | Internal tooling for RaiSE development |
| `agents/` | AI agent configurations | Agent prompts and configs |
| `research/` | Research notes | Internal research and exploration |
| `reports/` | Validation reports | Internal analysis and validation |
| `archive/` | Historical content | Deprecated or superseded content |

## Archive Subdirectories

- `archive/legacy-katas/` - Deprecated katas using old L0-L3 naming convention
- `archive/v1-framework/` - Original v1.x framework documentation

## Access

This content is:
- **Not** included in public documentation
- **Not** linked from README.md or public docs
- Available for internal reference only
