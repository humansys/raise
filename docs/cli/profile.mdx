View the developer profile stored at `~/.rai/developer.yaml`.

## `rai profile show`

Display the developer profile in YAML format. If no profile exists, shows a message guiding you to create one via `rai session start --name`.

```bash
rai profile show
```

No flags — displays profile and exits.

**See also:** [`rai session start`](/docs/cli/session/)
