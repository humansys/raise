Bienvenido a la documentación de RaiSE.

RaiSE combina una **metodología lean** con un **toolkit CLI determinístico** para hacer la ingeniería de software asistida por IA confiable, trazable y repetible.

## Qué encontrarás aquí

- **[Comenzar](getting-started.md)** — Instala el CLI y ejecuta tu primera sesión
- **[Conceptos](concepts/memory.md)** — Memoria, Skills, Gobernanza, Knowledge Graph
- **[Guías](guides/first-story.md)** — Tu Primera Historia, Configurar un Proyecto
- **[Referencia CLI](cli/index.md)** — Todos los comandos, flags y opciones

## Instalación rápida

```bash
pip install raise-cli
rai init
rai session start --project . --context
```
